# GitLink & GitHub Skills Web Services

基于 DeepSeek API + CLI 工具的开源项目智能分析服务。

- **GitLink 版**：使用 gitlink-cli，分析 GitLink 平台数据
- **GitHub 版**：使用 gh CLI，分析 GitHub 平台数据

## 部署要求

- Python 3.10+
- gh CLI（GitHub 版）或 gitlink-cli（GitLink 版）
- DeepSeek API Key（环境变量 API_KEY）
- GitHub Token（GitHub 版，环境变量 GH_TOKEN）
